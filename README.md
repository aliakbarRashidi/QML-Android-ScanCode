# QML-Android-ScanCode
QML for Android 实现二维码扫描功能（QML+QZXing）

使用 QtQuick control2实现界面，通过 Camera 调用摄像头，然后 QZXing 来解析二维码内容，最终实现了在 android 上的扫码功能。
